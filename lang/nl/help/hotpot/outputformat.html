Opmaak van de output

Je kunt de opmaak waarmee de test getoond wordt aanpassen.

beste
De test zal getoond worden met de beste opmaak voor de browser.
v6+
De test zal getoond worden in een klik en sleep opmaak voor v6+ browsers
v6
De test zal getoond worden in een opmaak voor v6 browsers
v5
De test zal getoond worden in een opmaak voor v5 browsers
v4
De test zal getoond worden in een opmaak voor v4 browsers
v3
De test zal getoond worden in een opmaak voor v3 browsers
Flash
Er zal Flash gebruikt worden om de test te tonen
mobiel
De test zal getoond worden met een opmaak die geschikt is voor het bekijken op kleine mobiele toestellen, zoals GSM's
