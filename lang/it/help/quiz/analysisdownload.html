Download analisi

I dati di analisi mostrati sullo schermo possono essere scaricati per ulteriori analisi. Sono disponibili tre diversi formati.
In ogni caso i dati saranno presentati sotto forma di tabella con titoli appropriati. Se la tabella è su più pagine, tutti i dati di tutte la pagine saranno scaricati in un unico file.

Formato foglio elettronico Excel

Si otterrà un foglio elettronico di tipo .xls.

Formato Writer OpenOffice

I dati saranno presentati tramite una tabella in un file testo OpenOffice di tipo .sxw.
Questo formato sarà disponibile solamente se si è installata la PHPWriter library nella sottocartella /moodle/lib.

Formato Testo

In questo caso i dati saranno scaricati come un singolo file di testo. Una riga per ogni riga della tabella con i dati separati da caratteri di tabulazione.
