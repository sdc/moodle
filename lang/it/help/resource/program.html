Risorse Programma

Una risorsa programma è una risorsa esterna, capace di accettare i dati passati da Moodle.

I dati vengono passati tramite HTTP GET (come parte dell'URL) e includono:

  • extern_nav: l'URL dal quale la risorsa è chiamata
  • extern_usr: il nome utente dell'utente attuale
  • extern_nam: il nome completo dell'utente attuale
  • extern_tim: l'ora attuale
  • extern_pwd: una password unica

Tutto questo sarà revisionato alcune volte nel prossimo futuro.
