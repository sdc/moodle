Pagina HTML

    Questo tipo risorsa permette facilemte di creare una pagina web completa all'inteno di Moodle, specialmente quando si usa l'editor visuale HTML (WYSIWYG).

    La pagina è salvata nella base dati, non in un file, è si ha molta libertà nell'utilizzare qualsiasi cosa ti piaccia utilizzando l'HTML, incluso Javascript.
