Pagpoformat ng teksto

Sa pagsusulat ng teksto sa Moodle, marami kang format na mapagpipilian upang malikha ang teksto mo, depende sa kakayanan mo at sa uri ng browser na ginagamit mo.

Kalimitan ay mapapabayaan mo na lamang ang setting na ito sa default nitong halaga at gagana naman ang lahat ayon sa inaasahan mo. .

1. Moodle auto-format

    Pinakamahusay ang format na ito kapag ang ginagamit mo ay normal na mga web form sa pagpapasok ng teksto (sa halip na Richtext HTML editor). Itype lang ang teksto nang normal, tulad nang pag nagpapadala ka ng email.

    Kapag isinave mo ang teksto mo, may ilang gagawin ang Moodle upang awtomatikong maiformat ang teksto mo.

    Halimbawa, ang mga URL na tulad ng http://yahoo.com o kaya'y www.yahoo.com ay makukumberte sa link.

    Pananatilihin ang mga line break mo, at ang mga blangkong liny ay magsisimula ng mga bagong parapo.

    Ang mga karakter na smiley tulad ng :-) ay awtomatikong makukumberte sa mga katumbas nilang larawan.

    Puwede ka ring mag-embed ng HTML code kung nais mo at mananatili ito.

Dagdag na kaalaman hinggil sa Moodle auto-formatting.

2. HTML format

    Inaakala ng format na ito na ang teksto ay purong HTML. Kung ginagamit mo ang HTML editor sa pag-edit ng teksto, ito ang default na format - lahat ng command sa toolbar ay lumilikha ng HTML para sa iyo.

    Kahit hindi mo gamitin ang Richtext HTML editor, maaari kang gumamit ng HTML code sa teksto mo at lalabas din naman ito nang ayon sa gusto mong mangyari.

    Hindi katulad ng Moodle auto-format, walang awtomatikong pagpoformat na nagaganap.

3. Plain text format

    Mahalaga ang format na ito kapag gusto mong maglagay ng maraming code o HTML na gusto mong madispley nang eksakto-eksakto sa isinulat mo.

    Nagkukumberte pa rin ito ng mga smiley at bagong linya, pero wala nang iba pang gagalawin sa teksto mo.

4. Markdown text format

    Ginagawa ng markdown format na gawing madali ang pagtype ng mga mahusay ang pagkakaformat na XHTML na pahina, sa pamamagitan lamang ng teksto na isinulat nang parang isinusulat ang isang email.

    Napakahusay nito para s apagsusulat ng malinis na pahina ng teksto na may ilang heading at ilang list pero walang gaanong link o larawan.

Dagdag na kaalaman hinggil sa Markdown format

5. Wiki text format

    Ang format na ito ay gumagamit ng mga ispesyal na code sa pagtatatype ng teksto, na kilala bilang paraang-Wiki. Kung alam na ninyo ang paraang-Wiki sa paglikha ng mga pahina at gusto mo itong gamitin sa kabuuan ng Moodle, ang format na ito ang gamitin mo.

    Tandaan na hindi ito tunay na Wiki, kaya't hingi kayo maaaring gumamit ng CamelCase sa paglikha ng bagong pahina. Maaari rin itong naiiba ng kaunti sa Wiki modyul sa Moodle, kaya't mag-ingat.
