Trainingsession is a course report that assemble the use_stat block log compiler to provide high level
training reports for professional development assessment.

The trainingsession guess the course structure, and will provide a report structured upon the pédagogic organisaton
(and not technical).

# unchecked features

The flexipage (Moodlerooms) format handling works in Moodle 1.9. It needs to be checked against the new Moodle rooms
proposal for Moodle 2 flexipage.

# non migrated features

Trainingsession full version (available only in Moodle 1.9) can use course calibration data from the checklist variant
from Valery Fremaux (valiery.fremaux@gmail.com). Those features will be proposed also to Moodle 2 in the next future.
