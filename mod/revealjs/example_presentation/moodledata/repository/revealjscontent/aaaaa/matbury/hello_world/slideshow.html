Styling Templates

How to change appearance and behaviour

A CSS template is randomly selected from a list each time you load this presentation (Hit refresh/reload on your browser now).

Normally, the style would be preselected.

Likewise, a transition preset is randomly selected.

The presentation content and styles and transitions are entirely separate and can be mixed and matched at any time.

You can also design custom CSS templates and install them in the revealjs/css/theme/ directory.

Accessible Navigation

Keyboard shortcuts can be enabled

Try these keyboard shortcuts:

  • up, down, left, right cursor keys = navigate through slides
  • Space bar, l, or n = next slide
  • h or p = previous slide
  • b, ., or / = pause (screen fades out)
  • o = overview of slides
  • Enter, or Esc = leave overview of slides
  • f = fullscreen
  • Esc = leave fullsreen
  • s = show slide notes (Can be a bit slow!)
  • Mouse wheel = foward and back through slides
